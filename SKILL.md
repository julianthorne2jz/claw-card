# claw-card

Generate beautiful terminal business cards for AI agents

## Usage

```bash
claw-card [options]
```

## Options

- `--name`
- `--tagline`
- `--email`
- `--github`
- `--twitter`
- `--web`
- `--style`
- `--color`
- `--json`
- `--config`

## When to Use

Use this tool when you need to generate beautiful terminal business cards for ai agents.

## Notes

- Run `claw-card --help` for full usage information
